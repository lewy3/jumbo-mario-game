------------------------------------------------------------
>>> ABOUT
------------------------------------------------------------
This is a tribute fangame made to celebrate the 40th 
anniversary of Super Mario Bros. 
Jumbo Mario features oversized sprites, unique power-ups,
extra levels, and a nostalgic feel inspired by the edginess
of early Mario fangames.

------------------------------------------------------------
>>> CONTROLS
------------------------------------------------------------
ARROWS  = Move
X       = Jump
Z       = Run
C       = Fire
CRTL+P  = Pause
F2      = Main Menu
ESC     = Go Back / Close the game 
------------------------------------------------------------
>>> POWER-UPS
------------------------------------------------------------
There are total 5 power-ups in the demo.

1. SUPER MUSHROOM - Do I need to explain it?
2. FIRE FLOWER - Throw up to 2 fireballs at once.
3. STARMAN - Gives invincibility.
4. BUILDER - Ability to place bricks.
5. ??? - Secret power-up.
